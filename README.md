# Student Management System

## 📌 Project Description

This is a full-stack web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).  
The application allows administrators to manage student records efficiently.

---

## 🚀 Features

- Add new student
- View all students
- Update student details
- Delete student
- REST API integration

---

## 🛠 Technologies Used

Frontend:
- React.js (Vite)
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose

---

## 🏗 Architecture Used

This project follows MVC Architecture:

- Model → Student Schema (server/models)
- View → React Frontend (client)
- Controller → Express route logic (server/index.js)

---

## ⚙️ How to Run the Project

### Backend Setup

1. Go to server folder:
   cd server
2. Install dependencies:
   npm install
3. Start server:
   npm start

Server runs on:
http://localhost:5000

---

### Frontend Setup

1. Open new terminal
2. Go to client folder:
   cd client
3. Install dependencies:
   npm install
4. Start frontend:
   npm run dev

Frontend runs on:
http://localhost:5173

---

## 🗄 Database

Make sure MongoDB is installed and running locally.
